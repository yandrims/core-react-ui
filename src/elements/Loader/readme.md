#### Usage

```markdown
import Loader from 'y-ui/dist/elements/Loader';
```

or

```markdown
import { Loader } from 'y-ui/dist/elements';
```

or

```markdown
import { Loader } from 'y-ui/dist';
```

#### Example

```js
import Loader from '../../elements/Loader';

<Loader />;
```

#### Example Color

```js
import Loader from '../../elements/Loader';

<Loader color="green" />;
```
